# Negative Results

This is the canonical register for failed mathematical approaches, disproved
engineering assumptions, invalid pruning rules, and completed experiments with
negative outcomes.

## Current register

No project negative result is accepted at bootstrap.

In particular, the following are not negative results:

- an upstream report that a historical `k = 14` parallel run exhausted memory;
- an unavailable or moved branch ref;
- a tiny smoke test that prints no counterexample;
- a candidate rejected by the independent verifier;
- a timeout or interrupted search.

Those facts, when observed, belong in upstream notes or run manifests. A future
entry here must include a stable ID, claim classification, exact scope,
reproducible evidence, limitations, and the accepting task/commit.
